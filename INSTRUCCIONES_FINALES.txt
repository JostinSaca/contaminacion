VISTAS CREADAS Y SUS FUNCIONES:
================================

1. DASHBOARD (Panel de Control)
   - Página de inicio
   - Información general del sistema
   - Guía de uso paso a paso

2. PRECARGA (Precarga de Datos)
   - Generación de datos históricos aleatorios
   - Inicialización de 5 zonas (Centro, Norte, Sur, Valle, Quitumbe)
   - Guardado en archivo binario: zonas.dat

3. HISTÓRICO (Datos Históricos)
   - Visualización de datos históricos en tabla
   - 30 registros de datos por mes
   - Filtros: Zona y Mes
   - Columnas: NA, NO2, SO2, CO2, PM2.5, Temp, Humedad, Viento

4. PREDICCIÓN (Nueva Predicción)
   - Ingreso de parámetros meteorológicos
   - Cálculo de predicción con factores ambientales
   - Generación de Índice de Contaminación (IC)
   - Guardado en archivo binario: predicciones.dat

5. RECOMENDACIONES (Historial de Recomendaciones)
   - Tabla con histórico de predicciones guardadas
   - Recomendaciones médicas personalizadas
   - Filtro por zona
   - Lectura de predicciones.dat

6. REPORTES (Generación de Reportes)
   - Dos tipos: Histórico y Actual
   - Exportación a archivo de texto: reporte_actual.txt
   - Visualización en editor integrado

PASOS PARA COMPILAR EN INTELLIJ IDEA:
======================================

1. Abre el proyecto en IntelliJ IDEA
   - File → Open → Selecciona la carpeta del proyecto

2. Espera a que IntelliJ indexe el proyecto
   - Esto puede tomar 1-2 minutos

3. Actualiza dependencias Maven
   - Click derecho en pom.xml → Maven → Reload projects
   - O: View → Tool Windows → Maven → Reload

4. Compila el proyecto
   - Build → Build Project (Ctrl+F9)
   - O: Maven → Lifecycle → clean → compile

5. Si hay errores:
   - Verifica que estén usando paquetes com.example.*
   - Asegúrate de tener Java 21 configurado
   - Limpia la caché: File → Invalidate Caches

ARCHIVOS DE DATOS GENERADOS:
=============================

El sistema crea los siguientes archivos en el directorio de trabajo:

• zonas.dat
  - Contiene: Zona[], Mes[], Dia[] serializado
  - Tamaño: ~500KB (aproximado)
  - Generado por: PrecargaView
  - Leído por: HistoricoView, PrediccionView

• predicciones.dat
  - Contiene: ResultadoPrediccion[] serializado
  - Tamaño: Crece con cada predicción
  - Generado por: PrediccionView
  - Leído por: RecomendacionesView, ReportesView

• reporte_actual.txt
  - Contiene: Reporte consolidado en formato texto
  - Leído por: Navegadores web, editores de texto
  - Generado por: ReportesView

• reporte_historico.txt
  - Contiene: Reporte histórico (30 días)
  - Generado por: ReportesView

NOTAS IMPORTANTES:
==================

1. Compatibilidad
   - El código mantiene 100% compatibilidad con la lógica CLI
   - No se modificó la lógica de negocio de SistemaCLI.java
   - Se reutilizan todos los algoritmos y cálculos

2. Persistencia
   - Los datos se guardan en archivos binarios en el directorio actual
   - Los archivos persisten entre sesiones

3. Validación
   - Los NumberFields validan rangos de entrada

4. Performance

   - Las operaciones de I/O se realizan de forma síncrona

5. Seguridad
   - La aplicación se ejecuta en localhost:8080
   - Acceso local únicamente
   - Preparada para agregar autenticación en el futuro
